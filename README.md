# ripe-range

Extract all CIDR Ranges for the Organization you want.

It use https://github.com/RIPE-NCC/whois/wiki/WHOIS-REST-API under the wood.

Installation
------------
```
$ npm install -g ripe-extract
```

-----
```
# show me microsoft IP blocks
```
